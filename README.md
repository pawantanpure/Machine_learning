# Logistic Regression Classification Project

## 📌 Project Description

This project implements a **Logistic Regression** model for binary classification using Python and Scikit-learn. The project preprocesses the dataset, generates predictions, evaluates the model using the **ROC-AUC** metric, and visualizes the **ROC Curve**.

---

## 🚀 Features

- Load dataset from a CSV file
- Data preprocessing using Pandas
- Convert categorical variables into numeric values
- Feature scaling using `StandardScaler`
- Predict class labels using Logistic Regression
- Save predictions to a CSV file
- Calculate ROC-AUC Score
- Plot ROC Curve

---

# Logistic Regression Classification with Future Prediction

## 📖 Overview

This project demonstrates how to build a **Logistic Regression** classification model using **Scikit-learn**. It includes data preprocessing, model training, evaluation, future prediction, and ROC curve visualization.

---

## 🚀 Features

- Load and preprocess dataset
- Split data into training and testing sets
- Standardize features using `StandardScaler`
- Train a Logistic Regression model
- Predict test data
- Evaluate model performance
- Generate confusion matrix
- Calculate accuracy score
- Display classification report
- Calculate training and testing accuracy
- Predict outcomes for future data
- Save future predictions to a CSV file
- Plot ROC Curve and calculate AUC Score

---
